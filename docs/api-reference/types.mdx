## Overview

PolygonKit provides comprehensive TypeScript types for all components, hooks, and utilities.

## Core Types

### Address

```typescript
type Address = `0x${string}`;
```

Ethereum address type with compile-time validation.

### PolygonKitConfig

```typescript
interface PolygonKitConfig {
  chains?: Chain[];
  projectId?: string;
  appName?: string;
  appDescription?: string;
  appUrl?: string;
  appIcons?: string[];
}
```

Configuration for PolygonKitProvider.

### Chain

```typescript
interface Chain {
  id: number;
  name: string;
  network: string;
  nativeCurrency: {
    name: string;
    symbol: string;
    decimals: number;
  };
  rpcUrls: {
    default: { http: string[] };
    public: { http: string[] };
  };
  blockExplorers?: {
    default: { name: string; url: string };
  };
}
```

## Hook Return Types

### UsePolygonKitReturn

```typescript
interface UsePolygonKitReturn {
  address: Address | undefined;
  isConnected: boolean;
  chain: Chain | undefined;
  balance: Balance | undefined;
  connect: () => void;
  disconnect: () => void;
  connectors: readonly Connector[];
}
```

```typescript
interface UsePolygonBalanceReturn {
```

## Type Guards

Check if a value is a valid address:

```typescript
function isAddress(value: string): value is Address {
  return /^0x[a-fA-F0-9]{40}$/.test(value);
}
```

## Generic Types

PolygonKit uses TypeScript generics for type-safe APIs:

```typescript
const { address } = usePolygonKit();
// address is Address | undefined

if (address) {
  // address is Address here
  const short = shortenAddress(address);
}
```
