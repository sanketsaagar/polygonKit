## TypeScript Support

PolygonKit is written in TypeScript and provides full type definitions:

```tsx
import type {
  Address,
  PolygonKitConfig,
  ConnectWalletProps,
  IdentityProps,
  TransactionButtonProps,
} from '@sanketsaagar/polygon-kit';
```
