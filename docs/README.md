# PolygonKit Documentation

This directory contains the Mintlify-powered documentation for PolygonKit.

## Setup

1. Install Mintlify CLI:
```bash
npm i -g mintlify
```

2. Run the docs locally:
```bash
mintlify dev
```

3. The documentation will be available at `http://localhost:3000`

## Structure

- `mint.json` - Mintlify configuration
- `introduction.mdx` - Landing page
- `quickstart.mdx` - Quick start guide
- `/installation` - Installation guides for different frameworks
- `/config` - Configuration documentation
- `/components` - Component documentation
- `/hooks` - Hooks documentation
- `/utilities` - Utilities documentation
- `/guides` - How-to guides
- `/api-reference` - API documentation

## Deployment

To deploy to production, connect your repository to Mintlify: https://mintlify.com
