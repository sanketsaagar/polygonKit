## The Fastest Way to Start

Use the `create-polygon-kit` CLI to scaffold a complete PolygonKit project instantly:

```bash
npx create-polygon-kit my-polygon-app
```

<Note>
  **Recommended**: The CLI is the fastest way to get started. It creates a complete project with pre-configured components, wallet connection, and a beautiful dashboard.
</Note>

## Quick Start with CLI

<Steps>
  <Step title="Run the CLI">
    Create your project with a single command:

    <CodeGroup>

    ```bash npm
    npx create-polygon-kit my-polygon-app
    ```

    ```bash yarn
    yarn create polygon-kit my-polygon-app
    ```

    ```bash pnpm
    pnpm create polygon-kit my-polygon-app
    ```

    </CodeGroup>

    The CLI will guide you through an interactive setup where you can:
    - Choose between Next.js or Vite
    - Select TypeScript or JavaScript
    - Optionally install dependencies automatically
  </Step>

  <Step title="Navigate to your project">
    ```bash
    cd my-polygon-app
    ```

    Your project includes:
    - Pre-configured PolygonKit components
    - Wallet connection with WalletConnect
    - Pre-built dashboard with Identity component
    - TailwindCSS v4 for styling
    - Full TypeScript support
  </Step>

  <Step title="Start the development server">
    Run your app:

    ```bash
    npm run dev
    ```

    Your app will be available at:
    - **Vite**: `http://localhost:5173`
    - **Next.js**: `http://localhost:3000`
  </Step>

  <Step title="Connect your wallet">
    Open your app in the browser and click "Connect Wallet" to see PolygonKit in action!

    The pre-built dashboard includes:
    - Beautiful welcome screen
    - Wallet connection with 300+ wallet support
    - User profile with Identity component
    - Avatar, address, and balance display
    - Responsive design with dark mode support
  </Step>
</Steps>

## Manual Installation (Optional)

If you prefer to add PolygonKit to an existing project manually, follow these steps:

<Steps>
  <Step title="Create a new project">
    Create a new React project (if you don't have one):

    <CodeGroup>

    ```bash npm
    npm create vite@latest my-polygon-app -- --template react-ts
    cd my-polygon-app
    ```

    ```bash yarn
    yarn create vite my-polygon-app --template react-ts
    cd my-polygon-app
    ```

    </CodeGroup>
  </Step>

  <Step title="Install dependencies">
    Install PolygonKit and required peer dependencies:

    <CodeGroup>

    ```bash npm
    npm install @sanketsaagar/polygon-kit wagmi viem @tanstack/react-query
    ```

    ```bash yarn
    yarn add @sanketsaagar/polygon-kit wagmi viem @tanstack/react-query
    ```

    </CodeGroup>
  </Step>

  <Step title="Set up the provider">
    Wrap your app with `PolygonKitProvider`:

    ```tsx src/main.tsx
    import React from 'react';
    import ReactDOM from 'react-dom/client';
    import { PolygonKitProvider } from '@sanketsaagar/polygon-kit';
    import App from './App';
    import './index.css';

    ReactDOM.createRoot(document.getElementById('root')!).render(
      <React.StrictMode>
        <PolygonKitProvider>
          <App />
        </PolygonKitProvider>
      </React.StrictMode>,
    );
    ```
  </Step>

  <Step title="Use PolygonKit components">
    Add components to your app:

    ```tsx src/App.tsx
    import { ConnectWallet, usePolygonKit } from '@sanketsaagar/polygon-kit';

    function App() {
      const { address, isConnected } = usePolygonKit();

      return (
        <div className="min-h-screen p-8">
          <h1 className="text-4xl font-bold mb-8">My Polygon App</h1>
          <ConnectWallet />
          {isConnected && <p className="mt-4">Connected: {address}</p>}
        </div>
      );
    }

    export default App;
    ```
  </Step>
</Steps>

## Common Use Cases

### Send a Transaction

```tsx
import { TransactionButton } from '@sanketsaagar/polygon-kit';

function SendButton() {
  return (
    <TransactionButton
      text="Send 0.001 MATIC"
      calls={[
        {
          to: '0xRecipientAddress',
          value: BigInt('1000000000000000'), // 0.001 MATIC
        },
      ]}
      onSuccess={(hash) => console.log('Transaction sent:', hash)}
      onError={(error) => console.error('Error:', error)}
    />
  );
}
```

### Display Token Balance

```tsx
import { TokenBalance } from '@sanketsaagar/polygon-kit';

function Balance({ address }: { address: string }) {
  return <TokenBalance address={address as `0x${string}`} />;
}
```

### Token Swap

```tsx
import { Swap } from '@sanketsaagar/polygon-kit';

function SwapWidget() {
  return (
    <Swap
      onSuccess={(hash) => console.log('Swap successful:', hash)}
      onError={(error) => console.error('Swap failed:', error)}
    />
  );
}
```

<Note>
  For production apps, make sure to configure your own [WalletConnect Project ID](/config/walletconnect).
</Note>
